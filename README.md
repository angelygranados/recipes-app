### Project based on the course [Building Applications in React and Redux](http://www.pluralsight.com/author/cory-house) on Pluralsight

This project allows the user to save its favorite cooking recipes, add new ones and edit them. It's a great and easy way to manage the traditional cooking handbooks.

## How to run it?

1. **Install [Node 8](https://nodejs.org)** or newer. Need to run multiple versions of Node? Use [nvm](https://github.com/creationix/nvm) or [nvm-windows](https://github.com/coreybutler/nvm-windows)(https://github.com/coryhouse/pluralsight-redux-starter/archive/master.zip)
2. **Navigate to this project's root directory on the command line.**
3. **Install Node Packages.** - `npm install`
4. **Start local project.** - `npm start`
